Un portfolio web unico che simula un ambiente desktop a finestre (Tiling Window Manager), ispirato all'estetica di Linux e Hyprland.

## 🚀 Caratteristiche
- **Desktop Virtuale**: Finestre ridimensionabili, trascinabili e con tiling automatico.
- **Integrazione GitHub**: Caricamento dinamico dei progetti tramite API di GitHub con filtri intelligenti.
- **Customizzazione**: Palette colori dinamica e selettore di wallpaper con effetti di sfocatura (blur) e luminosità.
- **Performance**: Single-page application leggerissima, realizzata in puro HTML/CSS/JS.

## 🛠️ Tecnologie
- Vanilla JavaScript (Custom Tiling Engine)
- CSS Variables per il sistema di temi
- GitHub API v3
- Estetica ispirata a Catppuccin e Hyprland
